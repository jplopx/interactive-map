# PetDex Interactive Map

**PetDex — Interactive Map** is a standalone module of the PetDex project, built to help users locate pet-related services around them.  
It includes a full interactive interface powered by Google Maps and Google Places, with real-time search, filters, routing, and dark mode.

## Current features
- Automatic geolocation  
- Address search  
- Categories:
  - Veterinarians  
  - Pet shops  
  - NGOs / animal shelters  
- Search radius selection (2 km, 5 km, 8 km)  
- Sorting by distance or rating  
- “Open now” filter  
- Route preview directly inside the map  
- External route opening in Google Maps  
- Dark/light mode with persistence  
- Paginated results from Google Places API  
- Result cards synchronized with map markers  
- Profile modal with detailed establishment information  

## About the project
The layout and front-end interaction are built using **HTML**, **CSS**, and **JavaScript**.  
The backend is powered by **Node.js**, **Express**, **CORS**, and **Helmet**, serving as a secure proxy for Google Places API.  
This module is independent and can be integrated into other systems, including the main PetDex platform.
